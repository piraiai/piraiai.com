Forsta Converter Questions

How do I convert a Word document to Forsta?

Upload your Word document to platform.piraiai.com, select Forsta as the output platform, choose bConvert, Word, or Forsta XML, review the converted survey in the workspace, and download the generated file. Import it into the Forsta portal using the path that matches your chosen format.

What file formats does Pirai AI generate for Forsta?

Pirai AI generates three Forsta-ready formats from a single upload: bConvert (.txt), Word (.docx), and Forsta XML (.xml). Each format has its own import path in the Forsta portal, so you can choose whichever fits your survey and access permissions.

What is Forsta bConvert format?

bConvert is a Forsta-specific plain text format used for survey import. Pirai AI generates a .txt file in bConvert syntax from any source document. Import via New Survey > More Options > Import File in the Forsta portal, no survey:build permissions required.

Do I need special permissions to import a Word document into Forsta?

Yes. The Forsta Word Importer requires survey:build permissions and is only available when creating a new survey. If you do not have these permissions, use the bConvert or Forsta XML output instead, both of which import via New Survey > More Options > Import File.

What Forsta XML version does Pirai AI support?

Pirai AI generates Forsta Plus XML compatible with v18 or later. Older XML versions are not accepted by Forsta and will prompt a system error on import. The generated XML also includes language translations where applicable.

Can I convert a Qualtrics QSF file to Forsta?

Yes. Upload your QSF file to platform.piraiai.com, select Forsta, choose bConvert, Word, or Forsta XML, and download the converted file. Pirai AI maps Qualtrics question types to their Forsta equivalents automatically.

Can I also convert to other platforms from the same upload?

Yes. From your Conversion History, click Download As on any completed conversion and select any of 26 supported platforms including Qualtrics, SurveyMonkey, REDCap, Google Forms, Microsoft Forms, Alchemer, LimeSurvey, and more. No reprocessing required.
