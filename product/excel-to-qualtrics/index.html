Does Pirai AI support skip logic and display logic from Excel?

Yes. Pirai AI reads skip logic and display logic from your Excel spreadsheet. You can specify logic in a dedicated column (e.g., "If Q1 = Yes, skip to Q5") or as text annotations alongside questions. Pirai AI converts these instructions to valid Qualtrics skip logic and display logic in the generated QSF file.

How should I format my Excel spreadsheet for best results?

Use columns for question number, question text, question type, and answer options. Put each question on its own row. List answer options in adjacent columns or as comma-separated values within a cell. Include a column for skip logic or display logic conditions if needed. Use separate sheets for different survey sections. Pirai AI handles most Excel layouts, but structured tabular data produces the best results.
