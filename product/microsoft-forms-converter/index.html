FAQ

Microsoft Forms Converter Questions

How do I import a Word document into Microsoft Forms?

Upload your Word document to platform.piraiai.com, select Microsoft Forms, and download the Quick Import-ready .docx file. In Microsoft Forms, click New Form, then the three-dot menu, then Quick Import. Upload your .docx file. Microsoft Forms creates all questions automatically.

What is Microsoft Forms Quick Import?

Microsoft Forms Quick Import is a built-in feature that reads a formatted Word (.docx) or PDF file and creates form questions automatically. It supports Choice, Text, Rating, Ranking, Number, and Date question types. The file must be correctly formatted to the Quick Import specification : Pirai AI generates this format automatically. Requires a Microsoft 365 work or school account.

Can I convert a Qualtrics QSF file to Microsoft Forms?

Yes. Upload your QSF file to platform.piraiai.com, select Microsoft Forms, and download the Quick Import .docx file. Import it into Microsoft Forms via New Form > three-dot menu > Quick Import. Pirai AI maps Qualtrics question types to Microsoft Forms question types automatically.

Is Quick Import available on all Microsoft Forms accounts?

Quick Import requires a Microsoft 365 work or school account. It is not available on personal Microsoft accounts (Outlook.com, Hotmail) or US Government Cloud accounts. The file must be .docx or .pdf format with a 10MB size limit.

What are the three flag types in Microsoft Forms Quick Import?

Microsoft Forms Quick Import flags three types of issues during review: (1) questions it could not parse and will skip, (2) questions it converted with reduced fidelity, and (3) informational notes about formatting choices. Pirai AI generates correctly formatted .docx files that minimise flag occurrences.
