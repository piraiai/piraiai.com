How do I convert a CSV file to Qualtrics?

Does Pirai AI support skip logic and display logic from CSV files?

Yes. Include a column in your CSV for skip logic and display logic conditions. Write instructions like "If Q1 = Yes, skip to Q5" or "Show Q3 only if Q2 = Other." Pirai AI reads these conditions and maps them to valid Qualtrics skip logic and display logic in the generated QSF file.

How should I format my CSV for best Qualtrics conversion results?

Structure your CSV with columns for question number, question text, question type, and answer options. Use one row per question. Separate answer options with commas in individual columns or use pipe delimiters within a single cell. Include a column for skip logic or display logic if needed. Save your CSV with UTF-8 encoding for proper handling of special characters. Pirai AI handles most CSV layouts automatically.
