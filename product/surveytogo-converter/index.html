FAQ

SurveyToGo Converter Questions

How do I import a Word document into SurveyToGo?

Upload your Word document to platform.piraiai.com, select SurveyToGo, review the converted survey in the workspace, and download the pre-formatted XML file. In SurveyToGo, navigate to your Project, right-click and choose Import, select SurveyToGo XML as the format, browse to your file, and click Import to generate the new survey.

What XML format does Pirai AI generate for SurveyToGo?

Pirai AI generates a SurveyToGo-compatible XML file using the Dooblo import schema (xmlns="http://www.dooblo.net/XMLImportSchema"). The XML includes question UUIDs, variable names, question types, render modes, answer options, and topics for matrix and grid questions.

Should I select SurveyToGo XML or Triple-S XML when importing?

Select SurveyToGo XML, not Triple-S XML. Triple-S is a separate format used for survey metadata and data transfer between platforms. Pirai AI output uses the SurveyToGo native Dooblo schema, which requires the SurveyToGo XML import option specifically.

Can I convert a Qualtrics QSF file to SurveyToGo?

Yes. Upload your QSF file to platform.piraiai.com, select SurveyToGo, and download the converted XML file. Pirai AI maps Qualtrics question types to their SurveyToGo XML equivalents automatically, including matrix, rank order, and constant sum questions.

Can I also convert to other platforms from the same upload?

Yes. From your Conversion History, click Download As on any completed conversion and select any of 26 supported platforms including Qualtrics, SurveyMonkey, REDCap, Google Forms, Microsoft Forms, Triple-S, and more. No reprocessing required.
