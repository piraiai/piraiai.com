FAQ

LSS Converter Questions

What is an LSS converter?

An LSS converter is a tool that converts survey documents (Word, PDF, Excel, CSV) into LimeSurvey LSS format for direct import into LimeSurvey. An LSS file is a LimeSurvey Structure file, an XML-based format that captures the complete survey structure including question groups, questions, answer options, and logic.

What is the difference between LSS DB version 5 and version 6?

LSS DB version 5 is compatible with LimeSurvey installations up to version 5.x. LSS DB version 6 is required for LimeSurvey 6.x and later. When converting with Pirai AI, select the version that matches your LimeSurvey installation in the platform selector after uploading your document.

How do I convert a Word document to an LSS file?

Upload your Word document to platform.piraiai.com, select LimeSurvey (DB version 5 or version 6), review the converted survey in the workspace, and click Finalize and Download. Import the .lss file into LimeSurvey via the + button on the Navigation bar > Import Survey > Drop file > Import survey.

Can I convert a Qualtrics QSF file to LimeSurvey LSS?

Yes. Upload your QSF file to platform.piraiai.com, select LimeSurvey (version 5 or version 6), and download the .lss file. Pirai AI maps Qualtrics question types (MC:SingleAnswer, MC:MultipleAnswer, Matrix, TE:Essay etc.) to their LimeSurvey equivalents (L, M, F, T etc.) automatically.
