How to Use the CSV Data Prep & Splitter Tool

From a single upload you can clean your data, reshape fields, extract a sample, and split into any number of output files. Everything runs in your browser. No file ever leaves your device.

Upload Your CSV File

Click the upload zone or drag and drop your CSV. The file is read entirely in your browser using PapaParse. Nothing is sent to a server at any point.

Clean & Transform (Optional)

Trim whitespace globally, randomise rows, apply per-column transforms (case formatting, prefix/suffix, find & replace, regex replace), concatenate fields into new columns, or add static value columns. Exclude any columns from output. A live 5-row preview shows changes as you type.

Choose an Export Mode

Export all records, sample a percentage or exact count, or split by row count, number of files, file size, or column value. A live preview shows the output before you commit.

Download as ZIP

Click Export. All output files are packaged into a single ZIP and downloaded to your device. Every file is UTF-8 encoded with a BOM, includes the header row, and uses CRLF line endings for maximum compatibility.

What Can You Do With This Tool?

From contact list prep to research data cleaning, these are the most common workflows this tool handles in one step.

Clean Data Before Platform Import

Trim accidental whitespace from ID and email fields, normalise case, and strip unwanted columns before importing into Qualtrics, Salesforce, HubSpot, or any CRM. One step replaces hours of manual Excel work.

Qualtrics SDS Preparation

Qualtrics Supplemental Data Sources have a 100,000-row cap and strict field requirements. Use the clean step to fix formatting, then split by row count to produce import-ready files within the limit.

Random Sampling for Research

Extract a random sample by percentage or exact count before a pilot study, A/B test, or quota check. The Fisher-Yates shuffle ensures a genuinely random cross-section, not just the first N rows.

Build New Fields from Existing Data

Concatenate FirstName and LastName into FullName, add a static BatchID or Wave column to every row, or use regex to reformat ID fields. New columns appear instantly in the column picker for review before export.

Segment by Region or Status

Split a contact list or survey export by Region, Status, or any column to produce one file per segment. Each file is named after the column value it contains.

Email and CRM Batch Imports

Many platforms cap list imports at 10,000 to 50,000 contacts. Clean the file first, then split by row count to produce ready-to-import batches that fit within your platform's upload limit.
