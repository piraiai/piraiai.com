What Can You Do With This CSV Cleaner?

Eight field operations plus global utilities. Stack as many as you need, in any order.

Trim whitespace

Remove leading and trailing spaces from every value in every column globally. The most common cause of mismatched records in joined datasets.

Format case

Apply UPPERCASE, lowercase, Title Case, or Sentence case to any column. Normalises contact names, region codes, and status values.

Add prefix or suffix

Prepend or append text to every value in a column. Useful for adding country codes, ID prefixes, or batch labels.

Find and replace

Exact string replacement across an entire column. Replace "N/A" with blank, fix known typos, or reformat date strings.

Regex replace

JavaScript regex find and replace with configurable flags (g, i, m). Use $1/$2 for capture groups. Invalid patterns leave values unchanged.

Concatenate fields

Combine two or more columns into a new column with a configurable separator. Classic use case: FirstName + LastName = FullName.

Static value column

Add a new column with the same value in every row. Useful for tagging records with a BatchID, Wave, Region, or Status before import.

Exclude columns

Choose exactly which columns appear in the output using the column picker. Uncheck any column to remove it. Works with original and new columns alike.
