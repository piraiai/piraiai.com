Overview

What are AI Editor Prompt Templates?

Standardized prompt patterns that help you communicate effectively with Pirai AI's survey editor. These templates ensure consistent, accurate results for common survey editing tasks - from simple text updates to complex matrix operations.

  • 100+ Templates - Covering all major survey editing operations
  • Copy-Paste Ready - Pre-formatted patterns you can use immediately
  • Time-Saving - Edit surveys in 30 seconds vs. 30 minutes manually
  • Consistent Results - Same pattern = same reliable output

How to Use This Guide

  1. Find your task - Browse categories or use Quick Start for common edits
  2. Copy the pattern - Click the copy button on any template
  3. Customize - Replace placeholders (Q9, question text, etc.)
  4. Paste & Send - Submit to AI Editor and watch it work

Formatting Rules

  • Multi-step prompts: Use numbered format 1), 2), 3)
  • Single-step prompts: No numbering needed
  • Action verbs: Update, Add, Remove, Combine, Convert, Move, Fill, Replace
  • Question references: Use format Q9, Q9-Q15, Q9 to Q15

üí° Best Practices

‚úÖ DO:

  • Be specific with question numbers (Q9, Q10-Q15)
  • Use standard action verbs (Update, Add, Remove, Combine, Convert)
  • Number steps when multiple actions needed
  • Specify question types (Matrix, Single Answer, Essay, Multiple Answer)
  • Reference exact text in quotes for updates
‚ùå DON'T:
  • Use vague references ("the first question", "that matrix")
  • Mix different numbering styles in one prompt
  • Skip question ID updates when reordering
  • Assume AI knows context without question numbers
