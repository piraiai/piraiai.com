Select your platform category, then choose your platform to see the full import steps.

⚠️

Important — Correct Answers: Pirai AI does not currently support setting correct answers during conversion. All questions default to the first option as correct. Before importing into your platform, open your downloaded file and update the correct answers for each question.

Qualtrics
Qualtrics supports two Pirai AI file types: QSF for a complete survey import, and Advanced Format (TXT/DOCX) for importing question content. Both use the same Tools → Import Survey path.
⬆ File Upload
Path 1: QSF Import
📥

Your Pirai AI file: Download the .qsf file from the platform. This creates a complete ready-to-use survey in your Qualtrics account.

  1. 1Log in to your Qualtrics account and navigate to the Survey tab of an existing or new project.
  2. 2Click Tools in the top toolbar, then select Import Survey from the dropdown.
  3. 3Click Browse and select your .qsf file downloaded from Pirai AI.
  4. 4Click Import. Qualtrics will load the survey and all questions and structure will appear in the Survey editor.
  5. 5Review the imported questions, add any logic or display conditions, then publish when ready.
⚠️

Importing a QSF through the Survey tab creates a new survey in your account. It will not overwrite an existing survey. QSF import is not available for Employee Experience project types (Engagement, Lifecycle, 360, etc.).

Path 2: Advanced Format (TXT / DOCX) Import
📥

Your Pirai AI file: Download the .txt or .docx Advanced Format file from the platform.

  1. 1In Qualtrics, open the Survey tab and click Tools → Import Survey.
  2. 2Select your .txt or .docx Advanced Format file and click Import.
  3. 3Qualtrics will parse the question structure and populate the survey editor. Review all imported questions for accuracy.
  4. 4Configure any display logic, skip logic, or embedded data fields as needed, then publish.

QuestionPro
QuestionPro supports direct Word document import, allowing you to upload your Pirai AI converted file and have questions populated automatically.
⬆ File Upload
📥

Your Pirai AI file: Download the .docx Word document file from the platform.

  1. 1Log in to your QuestionPro account and go to the Surveys page.
  2. 2Click New Survey, enter a survey name, and click Create Survey.
  3. 3In the blank survey editor, expand the Add Question menu and select Add from MS Word.
  4. 4On the pop-up, select your Pirai AI .docx file and click Import. QuestionPro will parse the document and create questions automatically.
  5. 5Review all imported questions, adjust question types if needed, and configure any logic or branching before publishing.
⚠️

The Add from MS Word option is only available for blank surveys. Make sure you start from a new, empty survey before attempting the import.

Google Forms
Google Forms does not support direct file import. Pirai AI generates a Google Apps Script file that automates the creation of your entire form. Open the Apps Script editor, paste the script, and run it to build your form automatically.
⚙ Apps Script
📥

Your Pirai AI file: Download the .txt file from the platform. It contains the complete Apps Script code pre-populated with all your survey questions and answer options.

⚠️

You will need a Google account to access Apps Script. The script creates a new Google Form automatically when executed. No manual question entry is required.

  1. 1Open your Pirai AI .txt file in any text editor. Select all the script content (Ctrl+A or Cmd+A) and copy it (Ctrl+C or Cmd+C).
  2. 2Go to script.google.com and sign in with your Google account. Click New project in the top-left corner.
  3. 3In the Apps Script editor, delete the default placeholder code (function myFunction() {}), then paste your copied Pirai AI script.
  4. 4Click the Save project icon (floppy disk) at the top of the screen to save your script.
  5. 5In the toolbar above your code, confirm the function dropdown shows createFormFromPiraiData. Then click the Run button (play arrow icon).
  6. 6Grant permissions (one-time only): Click Review permissions in the pop-up. Select your Google account. Click Advanced, then Go to project (unsafe). Click Allow.
  7. 7Check the Execution log at the bottom of the screen. Copy the form URL from the log and open it in a new tab to see your completed Google Form with all questions created automatically.

Microsoft Forms
Microsoft Forms has a native convert feature that imports Word or PDF documents directly and builds the form from your question content.
⬆ File Upload
📥

Your Pirai AI file: Download the .docx Word document file from the platform.

  1. 1Go to forms.office.com and sign in with your Microsoft account.
  2. 2Select Quick Import then Upload from this device.
  3. 3Select your Pirai AI .docx file from your device and choose Form or Quiz depending on your use case.
  4. 4Microsoft Forms will convert the file. The new form is autosaved under Recent and My Forms with the same title as the original document.
  5. 5When conversion is complete, select Start Review to check the results. Three types of feedback may appear: Missing content, Uncertain conversion, and Unsupported question type.
  6. 6For each flagged item, select the highlighted issue, resolve it, and click Resolved. Once all issues are addressed your form is ready to share.
⚠️

Microsoft Forms only supports Multiple Choice and Open Text question types on import. File size limit is 10MB. This feature is not available for US Government Cloud accounts.

REDCap
REDCap uses a Data Dictionary CSV upload to populate a project with fields and instruments. Pirai AI exports a CSV in the correct REDCap Data Dictionary format, ready to upload directly into a new project.
⬆ File Upload
📥

Your Pirai AI file: Download the REDCap Data Dictionary .csv file from the platform.

Step 1: Create a New Project
  1. 1Log in to REDCap and click New Project from the top menu.
  2. 2On the Create a New REDCap Project screen, enter a Project Title and select the Purpose from the dropdown.
  3. 3Under Project Creation Option, ensure Empty project (blank slate) is selected.
  4. 4Click Create Project.
⚠️

You must have permission to create projects in REDCap. If you do not have this access, contact your REDCap administrator. If you selected Research as the project purpose, complete all required fields before proceeding.

Step 2: Upload the Data Dictionary
  1. 1On the Project Setup tab, under Design your data collection instruments, click the Data Dictionary button.
  2. 2On the Data Dictionary tab, under Upload your Data Dictionary file (CSV file format only), confirm the correct Format for min/max validation values for date and datetime fields is selected from the dropdown.
  3. 3Click Choose File, browse to your Pirai AI .csv file, and select it. Click Open.
  4. 4Click Upload File. REDCap will validate the file and display a summary of the fields to be imported.
  5. 5Review the upload summary carefully. If no errors are shown, click Commit Changes to apply the Data Dictionary to your project.
  6. 6Once committed, you will see a Changes Made Successfully confirmation. From here you can go to the Online Designer to review instruments or continue in Project Setup.

Forsta
Pirai AI supports three Forsta file formats: bConvert (.txt), Word (.docx), and XML (.xml). Each uses a slightly different import path in the Forsta portal.
⬆ File Upload
Path 1: bConvert (.txt)
📥

Your Pirai AI file: Download the bConvert .txt file from the platform.

  1. 1In the Forsta portal, click New Survey.
  2. 2In the New Survey window, click More Options and select Import File.
  3. 3Drag and drop your .txt bConvert file into the box, or click Browse to locate the file on your device.
  4. 4Scroll down, enter a project name, and click Create Survey.
  5. 5Once uploaded successfully you will be redirected to the Survey Editor where you can review and edit your survey.
Path 2: Word Document (.docx)
📥

Your Pirai AI file: Download the Forsta Word .docx file from the platform.

⚠️

You must have survey:build permissions to access the Word Importer. The Word Importer is only available when creating a new survey.

  1. 1In the Forsta portal, click New Survey.
  2. 2In the New Survey menu, click the Start with Word tile.
  3. 3Drag and drop your .docx file into the file field, or click Browse to upload it from your device.
  4. 4Enter a name for your survey, then click Survey Options to configure language, Portal, and URL settings if needed.
  5. 5Click Create Survey to build your survey from the Word document.
Path 3: Forsta XML (.xml)
📥

Your Pirai AI file: Download the Forsta XML .xml file from the platform.

  1. 1In the Forsta portal, click New Survey.
  2. 2In the New Survey window, click More Options and select Import File.
  3. 3Drag and drop your .xml file into the box, or click Browse to locate the file on your device.
  4. 4Scroll down, enter a project name, and click Create Survey. The upload may take a few minutes for larger files.
  5. 5Once complete you will be redirected to the Survey Editor. If any elements were flagged, a survey load error will appear. Click Fix XML to review and resolve.
  6. 6Use the system warning tags to guide you through any required updates, then publish when all warnings are cleared.
⚠️

Forsta Plus XML import requires v18 or later files. Older versions will not be converted and will prompt a system error. XML files will include language translations where applicable.

UNICOM Intelligence
Pirai AI generates a UNICOM Intelligence MDD metadata file (.mdd) for use in mrStudio or UNICOM Intelligence Professional. This is the most feature-complete converter — no question types are skipped. Open the file directly in UNICOM Intelligence Professional (MDM Explorer) or load it through UNICOM Intelligence Author.
⬆ File Upload
📥

Your Pirai AI file: Download the UNICOM Intelligence .mdd file from the platform. Before opening, ensure the file is not marked as Read-Only in Windows Explorer.

⚠️

Ensure the .mdd file is not marked as Read-Only in Windows Explorer before opening. If it is, right-click the file, select Properties, and uncheck Read-Only. UNICOM Intelligence will throw an error and refuse to open a read-only file.

Method 1: UNICOM Intelligence Professional (MDM Explorer)

Use this method to inspect or modify variables, structure, or language settings inside the Metadata Model.

  1. 1Open UNICOM Intelligence Professional (or IBM SPSS Data Collection Studio).
  2. 2Go to the top menu and select File → Open.
  3. 3Change the file type dropdown filter to Metadata Document (*.mdd).
  4. 4Browse to your Pirai AI .mdd file, select it, and click Open.
Method 2: UNICOM Intelligence Author (Build Activity)

Use this method to load the file for continued scripting or questionnaire routing in the Build environment.

  1. 1Launch UNICOM Intelligence Author or open your local Interviewer Server Administration console.
  2. 2Select Open Existing Project or navigate to the Build activity tab.
  3. 3Browse to your Pirai AI .mdd file, select it, and open the project to begin scripting or routing.

  1. 1Click Compile or Update Survey to run the IntelliSurvey script parser.
