FAQ

Kahoot! Converter Questions

How do I convert a Word document to Kahoot?

Upload your Word document to platform.piraiai.com, select Kahoot!, and download the pre-formatted .xlsx file. In Kahoot!, click Create, then Add question > Import > Import spreadsheet. Upload the .xlsx file. Kahoot! validates and imports all questions automatically.

Can I convert a QSF file to Kahoot directly?

Yes. Upload your Qualtrics QSF file to platform.piraiai.com, select Kahoot!, and download the .xlsx file. Import via Kahoot! creator Add question > Import > Import spreadsheet. The conversion is direct: QSF in, Kahoot! .xlsx out. No intermediate Excel reformatting step required.

What happens to matrix questions when converting to Kahoot?

Kahoot! does not support matrix or grid questions. Pirai AI expands each matrix statement row into its own individual Quiz question, with the scale options (e.g. Strongly Agree, Agree, Neutral, Disagree, Strongly Disagree) as answer choices. Because Kahoot! has a 4-answer maximum, only the first 4 scale options are retained.

What is the Kahoot! answer option limit?

Kahoot! supports a maximum of 4 answer options per question. Pirai AI automatically caps all answer options at 4 during conversion. Questions with more than 4 options have the first 4 retained. Review the converted .xlsx file before importing if your questions have more than 4 options.

What question types are skipped when converting to Kahoot?

Open text questions, essay questions, descriptive blocks, and page breaks are skipped. They are listed in the conversion report so you know exactly what was excluded. Kahoot! only supports multiple choice and multi-select question types.
