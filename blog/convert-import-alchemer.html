How to Convert Word, PDF, or Excel into Alchemer

Arun Raj • April 29, 2026 • 2 min read

Alchemer has a built-in Import from Word feature that creates surveys from pasted text. The catch: the text must follow Alchemer's specific syntax rules — radio buttons need (), new pages need ::NewPage::, and open text needs underscores. Pirai AI generates all of that syntax automatically from any source document. You just copy and paste.

How Alchemer Import from Word Works

📋 Import method: Copy & Paste with Alchemer syntax

Unlike Qualtrics (which accepts a QSF file upload) or Zoho Survey (which accepts a .txt file upload), Alchemer's importer is paste-based. You copy the content from the Pirai AI .txt file and paste it directly into the Alchemer import field. Alchemer reads the syntax and builds the survey instantly.

What Pirai AI outputs for Alchemer

A .txt file containing your full survey formatted in Alchemer-compatible syntax. Radio buttons use (), checkboxes use [], page breaks use ::NewPage::, and open text uses underscores. Copy the content and paste it into the Alchemer import field — no syntax knowledge required on your end.
