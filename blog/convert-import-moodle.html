How to Convert Word, PDF, or Excel into Moodle (AIKEN Format)

Arun Raj • June 1, 2026 • 2 min read

Moodle uses the AIKEN plain text format for importing multiple choice questions into a quiz question bank. Creating AIKEN-formatted files manually means writing each question in exact syntax with no room for formatting errors. Pirai AI generates correctly formatted AIKEN .txt files automatically from any source document. Upload your Word, PDF, Excel, or CSV file and get a ready-to-import AIKEN file.

What Is AIKEN Format?

AIKEN is Moodle's plain text import format for multiple choice questions. Each question follows a strict structure: the question text on one line, each answer option labelled A. through Z. on separate lines, and the correct answer indicated by ANSWER: followed by the letter. Pirai AI generates this structure automatically from any source document.

Conversion Notes : Read Before Importing

Before you import, please note: Single and multiple choice questions convert natively. Multiple answer questions convert to single-answer with a warning. RankOrder and ConstantSum convert to multiple choice, losing ranking and sum functionality. Matrix questions are expanded: each statement becomes its own individual question with scale points as answer options. Open text and descriptive blocks are skipped and listed in the conversion report.
