How to Convert Word, PDF, or Excel into SurveyToGo

Arun Raj • June 8, 2026 • 2 min read

SurveyToGo is a professional CAPI/CAWI survey platform used by market research agencies and enterprise teams. It imports surveys via XML using the Dooblo import schema. Pirai AI generates a SurveyToGo-compatible XML file automatically from any source document : Word, PDF, Excel, CSV, QSF, or plain text. Upload your document, select SurveyToGo, and import the XML directly into your project.

How SurveyToGo Import Works

⬆ XML Import : SurveyToGo XML Format (Dooblo Schema)

SurveyToGo accepts survey structures via XML import using the Dooblo namespace (xmlns="http://www.dooblo.net/XMLImportSchema"). Pirai AI generates a fully structured XML file pre-formatted to this schema : including question UUIDs, variable names, question types, render modes, answer options, and topics for matrix and grid questions.

Your Pirai AI output for SurveyToGo

A .xml file formatted to the SurveyToGo Dooblo import schema. Import it directly via your Project > Import > SurveyToGo XML format.

Conversion Notes : Read Before Importing

Before you import, please note: When importing, select SurveyToGo XML as the format (not Triple-S XML). Triple-S is a separate format used for survey metadata and data transfer. The Pirai AI output uses the SurveyToGo native XML schema, not Triple-S.
