How to Convert Word, PDF, or Excel into Microsoft Forms

Arun Raj • April 29, 2026 • 2 min read

Microsoft Forms has one of the most capable native import features of any survey platform. It accepts Word documents and PDFs directly and builds the form automatically. The key is that your document needs to be structured correctly — and you need to know what to expect during the Start Review step. Pirai AI handles the document conversion so the import goes cleanly the first time.

What Microsoft Forms Accepts for Import

Microsoft Forms Quick Import accepts two file types directly:

FormatFile TypeBest For
Word document.docxStructured questionnaires, surveys
PDF.pdfFinalised forms, scanned questionnaires

Important limitations to know before importing:

Microsoft Forms natively supports Multiple Choice and Open Text on import. Complex types (matrix, multi-select, rank order, constant sum) are adapted by Pirai AI to the closest supported format. Descriptive text blocks and page breaks are skipped. File size limit is 10MB. Not available for US Government Cloud accounts.

What "adapted" means in practice

For question types that Microsoft Forms cannot replicate exactly, Pirai AI converts them to the closest available format rather than skipping them. A matrix question with 5 statements becomes 5 individual single-choice questions. A multi-select question becomes a single-choice question. This preserves your question content so you can adjust the type manually in the Microsoft Forms editor after import, rather than rebuilding from scratch.
