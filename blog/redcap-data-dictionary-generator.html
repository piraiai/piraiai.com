What Is a REDCap Data Dictionary?

A REDCap Data Dictionary is a CSV file that defines every field in a REDCap project. It has specific required columns: Variable / Field Name, Form Name, Field Type, Field Label, Choices (for radio, checkbox, dropdown), Text Validation Type, and more. REDCap reads this file and creates all instruments and fields in a single upload step rather than requiring you to add them one by one in the Online Designer.

How Pirai AI Generates a REDCap Data Dictionary

What Pirai AI does automatically

Pirai AI reads your source document, identifies question types and answer options, assigns correct REDCap field types (radio, checkbox, text, notes, dropdown), formats variable names to REDCap conventions (lowercase, no spaces, under 26 characters), codes answer choices in pipe-separated format (1, Option A | 2, Option B), and outputs a complete, upload-ready Data Dictionary CSV with all required columns populated.
