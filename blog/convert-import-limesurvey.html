How to Convert Word, PDF, or Excel into LimeSurvey

Arun Raj • June 8, 2026 • 2 min read

LimeSurvey is a free, open-source survey platform widely used by universities, research institutions, and NGOs worldwide. Its native import format is the LimeSurvey Structure file (.lss), an XML-based format that captures your complete survey structure. Pirai AI generates .lss files automatically from any source document. Upload your Word, PDF, Excel, CSV, or QSF file, select LimeSurvey, and import the .lss file directly into your LimeSurvey instance.

How LimeSurvey Import Works

⬆ File Upload : LimeSurvey Structure File (.lss)

LimeSurvey accepts survey structures via .lss file upload (XML-format, compatible with LimeSurvey 1.90 or later). Pirai AI generates a fully structured .lss file from your document : including question groups, questions, answer options, question type codes, and variable names. During import, LimeSurvey intelligently re-numbers all IDs automatically.

Your Pirai AI output for LimeSurvey

A .lss LimeSurvey Structure file formatted to the LimeSurvey XML import schema. Import it directly via the LimeSurvey Navigation bar plus button.

Conversion Notes : Read Before Importing

Before you import, please note: Single choice, multiple choice, free text, array/Likert, and ranking questions convert natively. Matrix questions are expanded: each row becomes its own individual question. Descriptive blocks convert to text display (X) type. RankOrder converts to ranking (R) type. Open text and essay questions convert to short (S) or long (T) free text. Review your converted survey in the LimeSurvey Online Designer after import.
