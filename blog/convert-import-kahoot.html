How to Convert Word, PDF, or Excel into Kahoot!

Arun Raj • June 1, 2026 • 2 min read

Kahoot! supports bulk question import via a pre-formatted Excel spreadsheet. Creating that spreadsheet manually means reformatting every question, option, and answer key by hand. Pirai AI generates the Kahoot! import spreadsheet automatically from any source document. Upload your Word, PDF, Excel, or CSV file and get a ready-to-import .xlsx file.

How Kahoot! Import Works

⬆ File Upload : Kahoot! Creator Import Spreadsheet

Kahoot! accepts question imports via a specific Excel template structure. The file must have the correct column headers, question types, and answer formatting. Pirai AI generates this file pre-formatted to Kahoot!'s required column structure : no manual reformatting needed.

Your Pirai AI output for Kahoot!

A .xlsx Excel file pre-formatted to Kahoot!'s import template. Upload it directly through the Kahoot! creator via Add question > Import > Import spreadsheet.

Conversion Notes : Read Before Importing

Before you import, please note: Single and multiple choice questions convert natively. Matrix questions are expanded: each statement becomes its own individual question. RankOrder and ConstantSum convert to single-answer questions. All questions are capped at 4 answer options (Kahoot!'s maximum). Open text, descriptive blocks, and page breaks are skipped and listed in the conversion report. Always review your converted file before importing.

    Log in to Kahoot! and open the creatorLog in to Kahoot! and click Create in the top-right corner to enter the Kahoot! creator.

    Import the spreadsheetClick Add question in the left panel, then select Import. At the bottom of the options, click Import spreadsheet.
