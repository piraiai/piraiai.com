How to Convert a Word Document to Microsoft Forms

Arun Raj • May 5, 2026 • 2 min read

You have a Word document with survey or quiz questions. You need them in Microsoft Forms. Unlike Google Forms, Microsoft Forms actually accepts Word document uploads directly via its Quick Import feature. The challenge is getting the import to go cleanly, and knowing what to do when flagged items appear. This guide covers all three methods and explains exactly what happens at each step.

Method 1: Manual Copy-Paste

This works for any file type and requires no external tools. For large surveys it is impractical.

  • 1
    Go to forms.office.com and create a new form Sign in with your Microsoft account at forms.office.com and click New Form.
  • 2
    Add each question manually from your Word document Click Add new for each question. Select the question type, paste the question text, and enter each answer option by hand.
  • 3
    Repeat for every question, then share For a 50-question survey this takes 3-4 hours. For 100 questions, plan for 6-8 hours. When done, click Send to share.

When to use it: You have fewer than 10 questions and no external tools. For anything larger, Method 2 or 3 is significantly faster.

Method 2: Native Quick Import (Word and PDF only)

Microsoft Forms has a built-in Quick Import feature that accepts Word documents and PDFs directly. It is the fastest method when your document is already well-structured. The catch is that complex formatting, embedded objects, and unsupported question types will appear as flagged items in Start Review.

  • 1
    Go to forms.office.com and select Quick Import Sign in at forms.office.com. On the home page, select Quick Import.
  • 2
    Upload from this device Choose Upload from this device. Select your Word (.docx) or PDF file. Choose Form or Quiz depending on your use case.
  • 3
    Wait for conversion and review Microsoft Forms converts the file and autosaves it under Recent and My Forms. Select Start Review to check the results and resolve any flagged items.

Understanding Start Review

After Quick Import, Microsoft Forms flags any content it could not convert cleanly. Three types of feedback appear: Missing content (content that was skipped entirely), Uncertain conversion (content Microsoft Forms is unsure how to handle), and Unsupported question type (types not supported on import such as matrix grids). Select each flagged item, make the correction, and click Resolved. The more structured your source document, the fewer flags appear.

Limitations: Quick Import only supports Word and PDF. It does not accept Excel, CSV, TXT, or XML. It is not available for US Government Cloud accounts. File size limit is 10MB.
