How to Convert Word, PDF, or Excel into REDCap

Arun Raj • April 29, 2026 • 2 min read

REDCap does not have a survey builder import wizard. It populates projects through a Data Dictionary CSV upload — a structured spreadsheet that defines every field, field type, and instrument in the project. Pirai AI generates that CSV automatically from any source document. Upload it to a new REDCap project and all your fields are created instantly.

What Is a REDCap Data Dictionary?

A REDCap Data Dictionary is a CSV file with specific column headers that REDCap reads to build your project's instruments and fields. It defines field names, field types (radio, checkbox, text, notes, dropdown), field labels, answer choices, and validation rules. Rather than building fields one by one in the Online Designer, uploading a Data Dictionary creates the entire project structure in a single step.

What Pirai AI outputs for REDCap

A .csv file in REDCap Data Dictionary format. Every column REDCap requires — Variable / Field Name, Form Name, Field Type, Field Label, Choices — is populated correctly. Upload the file via Project Setup > Data Dictionary > Upload File and click Commit Changes.

Convert Now

Step 1: Create a New REDCap Project

Permissions required: You must have permission to create projects in REDCap. If you do not have this access, contact your REDCap administrator. If you select Research as the project purpose, all required fields must be completed before proceeding.

  • 1
    Download your Data Dictionary CSV from Pirai AI Go to platform.piraiai.com, upload your document, select REDCap, and download the .csv Data Dictionary file.
  • 2
    Log in to REDCap and click New Project Click New Project from the top menu. On the Create a New REDCap Project screen, enter a Project Title and select the Purpose from the dropdown.
  • 3
    Select Empty project and click Create Project Under Project Creation Option, ensure Empty project (blank slate) is selected. Click Create Project.

Step 2: Upload the Data Dictionary

  • 1
    Go to Project Setup and open Data Dictionary On the Project Setup tab, under Design your data collection instruments, click the Data Dictionary button.
  • 2
    Confirm the date format and upload your CSV On the Data Dictionary tab, confirm the correct Format for min/max validation values for date and datetime fields is selected from the dropdown. Click Choose File, browse to your Pirai AI .csv file, and select it. Click Upload File.
  • 3
    Review the upload summary REDCap validates the file and displays a summary of all fields to be imported. Review the summary carefully. If no errors are shown, click Commit Changes to apply the Data Dictionary to your project.
  • 4
    Confirm success and review in the Online Designer A Changes Made Successfully confirmation will appear. From here, go to the Online Designer to review your instruments and fields, make any adjustments, and continue with your project setup.

Unique to REDCap: field-level structure

Unlike other platforms where questions map directly to survey items, REDCap organises content into instruments and fields. Pirai AI maps your source document questions to the correct REDCap field types, assigns them to instruments, and generates the full Data Dictionary structure — so the uploaded CSV works cleanly on the first commit.
