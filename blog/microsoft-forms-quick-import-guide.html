Microsoft Forms Quick Import: Complete Guide (Word, PDF, and Start Review)

Arun Raj • June 1, 2026 • 2 min read

Microsoft Forms Quick Import converts Word and PDF files into forms or quizzes at forms.office.com. In practice, the results depend heavily on how your source document is structured. This guide explains exactly how Quick Import works, what it supports, why it sometimes fails, and how to get the best results every time.

How Microsoft Forms Quick Import Works

Quick Import reads the layout of your Word or PDF file and attempts to identify questions, answer options, and response types. It creates a new form or quiz automatically and saves it under Recent and My Forms. After conversion, it takes you to a Start Review screen where any flagged items are shown for manual resolution.

How to access Quick Import

Go to forms.office.com and sign in with your Microsoft account. Select Quick Import on the home page. Choose Upload from this device. Select your Word (.docx) or PDF file. Choose Form or Quiz. Click Import. Select Start Review when conversion is complete.

Start Review Explained

After conversion, Microsoft Forms shows a Start Review screen with three types of feedback. Understanding each one tells you exactly what to fix:

Flag TypeWhat It MeansHow to Resolve
Missing contentContent from the source file was not importedAdd the missing question or text manually in the editor
Uncertain conversionMicrosoft Forms is unsure how to classify the contentCheck the question type assigned and correct if needed
Unsupported question typeThe question type is not supported on import (e.g. matrix)Rebuild as a supported type (Multiple Choice or Open Text)

Why Quick Import Sometimes Fails

The most common reasons Quick Import produces many flags or fails to import cleanly:

  • 1
    Complex formatting in the source documentTables, multi-column layouts, headers, footers, and embedded objects confuse the importer. Microsoft recommends arranging content vertically with questions and answer options in a simple inline format.
  • 2
    Unsupported question typesMatrix grids, Likert scales, ranking questions, and constant sum questions are not supported. They will be flagged and need to be rebuilt manually.
  • 3
    Scanned or image-based PDFsQuick Import cannot read text from image-based PDFs. Use Pirai AI to extract content from scanned documents first.
