How to Convert Word, PDF, or Excel into Forsta

Arun Raj • April 29, 2026 • 2 min read

Forsta gives you more import options than most platforms. Pirai AI supports all three: bConvert (.txt), Word (.docx), and Forsta XML (.xml). Each uses a different import path in the Forsta portal. Choose the format that matches your workflow and permissions.

Three Output Formats — One Conversion

File Upload

bConvert (.txt)

Standard Forsta import. New Survey > More Options > Import File. Simplest path for most surveys.

File Upload

Word (.docx)

New Survey > Start with Word. Requires survey:build permissions. New surveys only.

File Upload

Forsta XML (.xml)

New Survey > More Options > Import File. Best for complex surveys and language translations. Requires v18+.

Convert Now

Path 1: bConvert (.txt) — Standard Import

  • 1
    Upload your document to Pirai AI and select bConvert output Go to platform.piraiai.com, upload your file, select Forsta, and choose bConvert (.txt) as the output format. Download the .txt file.
  • 2
    In the Forsta portal, click New Survey In the New Survey window, click More Options and select Import File.
  • 3
    Upload your .txt file Drag and drop your bConvert .txt file into the box, or click Browse to locate it on your device.
  • 4
    Enter a project name and click Create Survey Scroll down, enter a project name, and click Create Survey. On success you will be redirected to the Survey Editor to review and edit your survey.

Path 2: Word (.docx) — Start with Word

Permissions required: You must have survey:build permissions in Forsta to access the Word Importer. The Word Importer is only available when creating a new survey.

  • 1
    Upload your document to Pirai AI and select Word output Go to platform.piraiai.com, upload your file, select Forsta, and choose Word (.docx) as the output format. Download the .docx file.
  • 2
    In the Forsta portal, click New Survey then Start with Word In the New Survey menu, click the Start with Word tile.
  • 3
    Upload your .docx file Drag and drop your .docx file into the file field, or click Browse to upload it from your device.
  • 4
    Enter a name and click Create Survey Enter a name for your survey, then click Survey Options to configure language, Portal, and URL settings if needed. Click Create Survey to build from the Word document.

Path 3: Forsta XML (.xml) — Full Import

Version requirement: Forsta Plus XML import requires v18 or later files. Older versions will not be converted and will prompt a system error. Pirai AI generates XML at the correct version. XML files include language translations where applicable.

  • 1
    Upload your document to Pirai AI and select XML output Go to platform.piraiai.com, upload your file, select Forsta, and choose Forsta XML (.xml) as the output format. Download the .xml file.
  • 2
    In the Forsta portal, click New Survey > More Options > Import File Drag and drop your .xml file into the box, or click Browse. Enter a project name and click Create Survey. Larger files may take a few minutes to process.
  • 3
    Resolve any flagged items and publish If any elements were flagged, a survey load error will appear after import. Click Fix XML to review. Use the system warning tags to guide you through any required updates. Publish when all warnings are cleared.
